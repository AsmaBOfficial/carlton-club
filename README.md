# Carlton & District Liberal Club — Website

React (Vite) + Tailwind CSS v4 + React Three Fiber (3D snooker-break hero).

## Run locally
```
npm install
npm run dev
```

## Build
```
npm run build
```

## Deploy to Vercel
1. Push this repo to GitHub.
2. Go to vercel.com → New Project → Import the GitHub repo.
3. Framework preset: Vite (auto-detected). No env vars needed.
4. Deploy.

## Notes
- Photography is placeholder (picsum.photos) — swap for real club photography before public launch (min. 20 hi-res images per brief).
- Membership pricing is indicative — confirm with the club before launch.
- `prefers-reduced-motion` is respected: the 3D hero falls back to a static gradient.
